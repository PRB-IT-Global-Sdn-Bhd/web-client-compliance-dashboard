# Web App — Client & Stakeholder Portal / Compliance Dashboard

> **PRB IT Global Sdn Bhd** | Custom Software for SME Engineering & Construction Clients (Malaysia)

## Overview

A secure, role-based **Client and Stakeholder Portal** combined with a **Compliance Dashboard** that provides real-time project visibility to clients, investors, government liaisons, and internal management. The platform ensures transparency, accountability, and regulatory adherence throughout the project lifecycle.

## Key Features

| Module | Description |
|---|---|
| Stakeholder Portal | Personalised dashboards per client/stakeholder role |
| Project Progress | Live status updates, milestone completion, photo logs |
| Compliance Tracker | CIDB, DOSH, ISO, and QLASSIC compliance checklists |
| Document Sharing | Secure upload and download of drawings, reports, contracts |
| Notification System | Email/SMS alerts for approvals, deadlines, and incidents |
| Audit Trails | Immutable activity logs for regulatory audits |
| Reporting | Automated compliance and progress reports |

## Technology Stack

- **Frontend:** React.js, TypeScript, TailwindCSS
- **Backend:** Node.js / NestJS
- **Database:** PostgreSQL
- **Auth:** SSO / OAuth2 / Role-Based Access Control (RBAC)
- **Hosting:** AWS / Azure (Malaysia region)

## Target Users

Project owners, clients, government regulators, QS consultants, and project directors.

## Getting Started

```bash
git clone https://github.com/PRB-IT-Global-Sdn-Bhd/web-client-compliance-dashboard.git
cd web-client-compliance-dashboard
npm install && npm run build
npm start
```
